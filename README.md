# kkaemiGG
온라인 게임 리그오브레전드 전적 검색 사이트 입니다.

* Spring Initializr를 사용해서 프로젝트를 생성했습니다.

  - spring 2.3.4
  - JAVA 11
  - template engine으로 Thymeleaf를 사용했습니다.

  이 프로젝트를 진행하면서 JSON을 JAVA객체로 바꿔주는 JACKSON과 RestTemplate에 대해서 알게되었습니다.
